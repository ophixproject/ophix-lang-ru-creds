# ophix-lang-ru-creds

> Русский языковой пакет для ophix-creds.

Russian translation pack for [ophix-creds](https://ophix.io).

## Install

```bash
pip install ophix-lang-ru-creds
```

Then set `LANGUAGE_CODE = "ru"` in your server `.env`.

## What is translated

- Admin UI strings: app name, inline labels, column headers, fieldset titles
- Help text strings from `ophix-server-base` (ClientArtifactBase fields)
- Documentation pages: credentials guide, legacy upgrade guide
